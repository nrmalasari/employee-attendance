# AttendFlow - Admin Absensi Karyawan

Sistem administrasi absensi karyawan berbasis web dengan fitur CRUD lengkap. Dibangun menggunakan **Backstrap** (Bootstrap 5) dan **LocalStorage** untuk penyimpanan data.

## 📋 Fitur Lengkap

### Halaman Dashboard (index.html)
- ✅ **List Data Karyawan** - Menampilkan semua data absensi dalam tabel modern
- ✅ **Update Data** - Edit data karyawan dengan modal popup
- ✅ **Delete Data** - Hapus data dengan konfirmasi modal
- ✅ **Sort By** - Urutkan berdasarkan Nama, Tanggal, atau Jam Masuk
- ✅ **Pagination** - 5 data per halaman dengan navigasi
- ✅ **Search** - Cari karyawan berdasarkan nama
- ✅ **Filter Gender** - Filter data Laki-laki/Perempuan
- ✅ **Filter Tanggal** - Filter berdasarkan tanggal absen
- ✅ **Statistik** - Card total hadir, hadir hari ini, persentase, total karyawan

### Halaman Input Absensi (add.html)
- ✅ **Nama Lengkap** - Input teks dengan placeholder
- ✅ **Alamat** - Textarea dengan contoh format
- ✅ **Jenis Kelamin** - Dropdown pilihan
- ✅ **Tanggal Absen** - Date picker
- ✅ **Jam Masuk** - Time picker (format 24 jam)
- ✅ **Jam Keluar** - Time picker (format 24 jam)
